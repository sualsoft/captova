Captova AI Submarine workstation is an IDP solution which is totally isolated, bare-metal, off-the-grid on- Captova AI Hyper workstation is a hybrid IDP solution which has the same architecture as Captova AI Submarine except for the fact that it can be hosted as premise system for top secret documents; it is not connected to any network or web service.

The Submarine workstation does not need to be connected to a network because the AI Models reside within the Submarine itself, giving it an ultra-high level of security.

For privacy and security reasons, the archive for documents also resides inside the Submarine workstation itself. However, the customer can always backup the archives to a storage facility of his choice or transmit the data via an API to his own repository. The Submarine on Bare Metal provides the highest level of Captova’s Privacy–Accuracy–Speed–Security (P.A.S.S) features. 		Captova AI Hyper workstation is a hybrid IDP solution which has the same architecture as Captova AI Submarine except for the fact that it can be hosted asbare-metal server in a Virtual Private Cloud (VPC) at any cloud service of customer’s choice (such as Google Cloud Platform, Azure, or AWS). The Hyper workstation can also reside on-premise if the customer so chooses. In this case, the workstation will access AI models from Captova’s servers on Google Cloud Platform (GCP) taking full advantage of Captova’s pre-trained AI Models that are already created for others (exponential footprint benefits). Since the all the AI Models are in machine codes with no private data or documents, Captova Hyper still benefits from Captova’s P.A.S.S. features.
